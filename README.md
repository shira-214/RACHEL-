# המשכיר — פרויקט גמר

מערכת לניהול והזמנת דירות לשכירות קצרה.  
ארכיטקטורה: **WPF** (לקוח) + **WCF** (שרת) + **Entity Framework 6** + **LocalDB**.

## הרצה

1. הפעילי **Host.exe** (חייב לרוץ לפני ה-WPF):
   ```
   RacheliProject\Host\bin\Debug\Host.exe
   ```
2. הפעילי **WPF**:
   ```
   WpfRentingApartementRacheli\WpfRentingApartementRacheli\bin\Debug\WpfRentingApartementRacheli.exe
   ```

**WSDL (בדיקה):**  
http://localhost:8733/Design_Time_Addresses/Server/Service1/?wsdl

## נתיבים

## זרימות עיקריות

### שוכר
Landing → כניסת שוכר → חיפוש דירה → הזמנה (+ אשראי) → ההזמנות שלי

### משכיר
Landing → כניסת משכיר → לוח בקרה → הוספת דירה / תמונה → צפייה בהזמנות

### מנהל
Landing → כניסת מנהל → אזור ניהול → CRUD על 10 טבלאות

## הערות

- מחיקת רשומה עם תלויות (FK) תציג הודעה בעברית — זו התנהגות תקינה.
- סדר מחיקה מומלץ: הזמנות → תמונות → תוספות-דירה → דירה.
- דפים `LookForApartement`, `AddApartement` — legacy, לא בשימוש.
